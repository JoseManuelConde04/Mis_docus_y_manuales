#Comandos de linux que podemos instalar que son útiles para gestionar el sistema y navegar por la cli o consola  
1. *fzf*: como find pero más fácil de usar
2. *tldr*: un man mejor que help
3. *ack*: un cambio de grep, para búscar lo que realmente haga falta 
4. *mtr*: es un ping y traceroute junto
5. *eza*: es un ls bonito y más visual
6. *batcat*: sustituye a cat de manera que lo hace más bonito visual y sencillo de usar
7. *ripgrep*: para búscar entre directorios 
8. *gping*: ping pero con gráfico bonito  
## Herramientas que suelo instalar en mis servidores linux para homelabs  
- ```htop``` o en su defecto ```btop```, de las mejores herramientas para ver procesos matarlos etc... Es literalmente igual que el administrador de tareas de windows
- ```ncdu``` un comando que sirve para analizar el espacio en disco, analizar cuantos logs hay en el sistema, que herramienta o que hay gastando más espacio del normal y se le pueden borrar facilmente
- ```ufw``` es el firewall más sencillo de usar
- ```fail2ban``` es un monitor de logs files al uso bloqueando automáticamente direcciones ip que pueden ser malignas, pudiendo aplicar firewall rules cuando hay sospechas de actividad no deseada.
- ```unattended-pugrades``` para instalación desatendida de actualizaciones de seguridad y estabilidad
- ```Docker``` es lo mejor en cuanto a creación de contendores
